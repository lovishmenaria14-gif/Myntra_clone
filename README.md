# 🛍️ Myntra Clone

A responsive front-end clone of the Myntra homepage built using **HTML5** and **CSS3**. This project was created to practice modern web development concepts such as Flexbox, Grid Layout, responsive design, and UI recreation.

---

## 🚀 Features

- Responsive Navigation Bar
- Search Bar
- Wishlist & Bag Section
- Hero Banner
- Sale Banner Section
- Product Category Cards
- Footer with Popular Searches
- Clean and Modern UI
- Hover Effects

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid

## 🔧 Setup Instructions

1. Clone the repository

```bash

```

2. Open the project folder

```bash

```

3. Run the project

Simply open:

```bash
index.html
```

in your browser.
